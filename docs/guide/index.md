# Introduction

Welcome to the documentation!

This is a simple VitePress starter template to help you get started quickly.

## What is VitePress?

VitePress is a static site generator (SSG) designed for building fast, content-centric websites. It takes your Markdown content, applies a theme, and generates static HTML pages.

## Features

- **Fast Hot Reload** - See your changes instantly
- **Vue-Powered** - Use Vue components in Markdown
- **Built-in Search** - Full-text search out of the box
- **Responsive** - Mobile-friendly by default
